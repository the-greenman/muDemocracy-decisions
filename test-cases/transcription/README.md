# Transcription Fixtures

These fixtures are sampled from a real Whisper transcription run and are intended for local tests (T0/T1 prep).

This README is fixture-local only. For the transcription service and architecture context, use `../../apps/transcription/README.md` and `../../docs/transcription-architecture.md`.

## Files

- `whisper-output.sample.json`: Combined sample containing mapped `events` and a trimmed Whisper `rawResponse`
- `whisper-verbose-json.sample.json`: Whisper `verbose_json` payload sample (trimmed to first 20 segments)
- `whisper-events.sample.json`: Mapped transcript events sample (first 20 events)

## Source

Derived from local output generated by:

```bash
set -a; source .env; set +a
pnpm --filter @repo/transcription exec node dist/index.js transcribe-local \
  "examples/9 Dec at 21-45 - party utility tasks.m4a" \
  --output /tmp/transcription-local-examples.json
```

## Regenerate Samples

```bash
mkdir -p test-cases/transcription
jq '{events: (.events[:20]), rawResponse: (.rawResponse | .segments = (.segments[:20]) | .text = ((.segments | map(.text) | join(" ")) | sub("^\\s+"; "")))}' \
  /tmp/transcription-local-examples.json > test-cases/transcription/whisper-output.sample.json

jq '.rawResponse | .segments = (.segments[:20]) | .text = ((.segments | map(.text) | join(" ")) | sub("^\\s+"; ""))' \
  /tmp/transcription-local-examples.json > test-cases/transcription/whisper-verbose-json.sample.json

jq '.events[:20]' /tmp/transcription-local-examples.json > test-cases/transcription/whisper-events.sample.json
```
